Getting Started !

This page is a demo for FF4j, Feature Flipping for Java. Items listed below are driven by features.
  • You can EVALUATE if a feature is toggled (here AwesomeFeature)
  • You can READ a feature in the UI like the description
  • You can READ a property in the UI like the value
